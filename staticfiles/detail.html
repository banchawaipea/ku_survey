บทคัดย่อ

โครงงานนี้เป็นการออกแบบแพลตฟอร์มวิเคราะห์และประมวลผลภาพถ่ายทางอากาศเพื่อการเกษตรกรรม โดยมีความสามารถในการกระจายแนวกว้างผ่านระบบ Kubernetes เป้าหมายของโครงงานนี้คือเพื่อต้องการเพิ่มประสิทธิภาพการทำงานของระบบ WebODM ให้สามารถรองรับจำนวนผู้ใช้งานได้มากขึ้น โดยทดสอบปริมาณผู้ใช้งานผ่านการใช้ Apache JMeter วัดค่า response time ต่อจำนวน active thread และวัดค่าจำนวนการตอบกลับต่อวินาที ด้วยการออกแบบใหม่ของโครงงานนี้ทำให้ response time สูงขึ้นร้อยละ 1.9 แต่ว่ารองรับปริมาณการใช้งานได้สูงกว่าระบบ WebODM เดิม ร้อยละ 101 และมีผลการประมวลผลภาพถ่ายที่ทั้งเร็วขึ้นกว่าระบบเดิม 4.58% ที่ชุดภาพถ่ายจำนวน 310 ภาพ และช้าลงกว่าระบบเดิม 36.42% ที่ชุดภาพถ่ายจำนวน 756 ภาพ

Abstract

This thesis describes the design of a platform for agricultural drone image processing and analysis which can scale out on a Kubernetes cluster. The primary objective of this thesis is to enhance WebODM's performance in order to support a more significant number of users. With Apache JMeter, this thesis benchmarked the response time per active thread and the number of responses per second. In comparison to the original WebODM, this new design has a 1.9 percent longer response time. Nonetheless, it increases throughput up to 101 percent over the original WebODM’s with some difference in the drone image processing result, 4.58% faster than the original WebODM’s on 310 images dataset, and 36.42% slower on 756 images dataset.
